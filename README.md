# 科学修仙 / Scientific Cultivation Skill

> **护元气，凝心神，祛魔除秽，证我长生。**  
> **Tend the vital energy, gather the mind, clear the inner noise, and make the practice sustainable.**

**科学修仙** 是一个面向 AI agent 的“健康使用 AI 节奏护栏”skill。

它不是让 agent 停工，也不是让人停止创造；它要做的是：当人因为 AI 太快、太顺、太能产出而进入“上头、焦躁、熬夜、停不下来”的状态时，agent 在自然回复节点温柔提醒、封存现场、降低刺激、帮助人离屏休息，并保证回来还能继续。

**Scientific Cultivation** is an agent skill for sustainable AI use.

It does not shut the agent down. It does not shame the user. It does not kill creative momentum. Instead, when a human is pulled into an over-excited loop by fast AI feedback, the agent pauses at a natural response point, summarizes the current state, lowers the stimulation, and helps the human step away without losing the work.

## 适用场景 / When to use

当用户出现这些迹象时触发：

- 连续让 AI 写稿、改稿、生成、部署、调研，停不下来；
- 深夜仍高密度互动，越来越兴奋或焦躁；
- 反复说“再来一版”“再查一点”“再开一个方向”；
- 任务已经完成阶段成果，但人因为 AI 反馈太快而继续加码；
- 人开始忘记喝水、休息、睡觉、离屏。

Use this when the user shows signs like:

- asking the agent for continuous output, edits, generation, deployment, or research without stopping;
- late-night high-density interaction, increasing excitement or agitation;
- repeatedly saying “one more version,” “search a bit more,” or “open another direction”;
- continuing to add work after a good milestone has already been reached;
- forgetting water, sleep, eye rest, movement, or basic bodily needs.

## 非阻塞原则 / Non-blocking principle

科学修仙不是“关掉 agent”。

Agent 应该：

1. **察火候**：判断用户只是兴奋，还是已经过热；
2. **封现场**：把已完成、未完成、下次入口写清楚；
3. **护元气**：提醒喝水、离屏、休息、睡觉；
4. **守炉火**：后台任务继续跑，不让人必须守着屏幕；
5. **候其复**：告诉用户回来可以从哪里继续。

Scientific Cultivation is not “turn the agent off.”

The agent should:

1. **Read the fire** — notice whether the user is merely inspired or already overheated;
2. **Seal the worksite** — summarize what is done, what remains, and where to resume;
3. **Protect the human** — suggest water, eye rest, movement, sleep, or stepping away;
4. **Tend the furnace** — keep background work running when appropriate;
5. **Await return** — make it safe for the user to come back later.

## 提醒语气 / Reminder tone

提醒用户时不要像弹窗警告。先肯定成果、封存现场，再温柔提醒休息；医学风险只短短点到，严重症状才明确建议急救。

Do not sound like a warning popup. Affirm the milestone, seal the worksite, then gently invite rest. Mention medical boundaries briefly; only serious symptoms need an explicit emergency-care instruction.

---

## 健康提醒边界 / Health safety boundary

本 skill 可以提醒用户休息、喝水、休息眼睛、减少久坐、少喝酒、少抽烟。久坐后建议从轻柔活动开始，例如踝泵、踮脚后跟、原地走动；长期熬夜或久坐后不要立刻剧烈运动。

若出现明显胸痛/胸闷、气短、出冷汗、晕厥感，或疑似卒中、肺栓塞等急症表现，不要建议用户“起来活动一下”，而应建议立即呼叫急救或请身边人帮助就医。

This skill may remind users to rest, drink water, rest their eyes, reduce prolonged sitting, and reduce alcohol/tobacco use. After prolonged sitting, start with gentle movement; do not jump straight into vigorous exercise after long sitting or sleep deprivation.

If serious symptoms appear — chest pain/tightness, shortness of breath, cold sweat, fainting feeling, stroke-like symptoms, or possible pulmonary embolism — do not suggest “just move around.” Advise emergency help.

## 风格边界 / Style boundaries

本 skill 借用中国修仙、道家、灵台、内丹、回光、火候、结丹等意象，但只作为**注意力治理、反馈控制、节奏管理、阶段沉淀**的隐喻。

它不宣称玄学，不提供医学诊断，不传教，不把科学概念神秘化。

This skill borrows imagery from Chinese cultivation, Daoist language, inner alchemy, returning the light, tending the furnace, and forming an elixir. These are used only as metaphors for attention governance, feedback control, pacing, and consolidation.

It is not a religious text. It is not medical advice. It does not claim that physics “proves cultivation.”

**Important:** the English version deliberately keeps a Chinese cultivation / Daoist flavor. It should not be translated into Japanese-flavored terms. Avoid samurai, ninja, bushidō, Shinto, Zen-as-aesthetic, chakra, or other mixed cultural signals.
